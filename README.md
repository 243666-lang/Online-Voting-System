# Online Voting System

## Overview
A console-based Online Voting System built in C++ that allows voters to cast votes and admins to manage elections.

## Features
- Voter registration
- Cast vote (one vote per voter)
- Admin login
- View results
- Register new candidates
- File-based storage (voters, admins, votes)
