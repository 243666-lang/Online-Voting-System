# Online Voting System

## 📌 Overview

A console-based **Online Voting System** built in **C++** that allows voters to cast votes for candidates and admins to manage the election process. Implements file-based persistence for storing voters, admins, and vote counts.

## 🎯 Features

### For Voters
- Register as a new voter
- Cast vote for a candidate (one vote per voter)
- View election results

### For Admins
- Register as admin
- Login with admin ID
- View real-time voting results
- Register new candidates

## 🛠️ Technologies Used

| Category | Technology |
|----------|-----------|
| Language | C++ |
| Storage | File I/O (`.txt` files) |
| Concepts | OOP (Inheritance, Polymorphism), File Handling |
